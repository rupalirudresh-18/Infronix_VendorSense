
# 🚀 VendorSense – Intelligent Vendor Risk & Decision System

VendorSense is an AI-powered decision support system that helps businesses evaluate, analyze, and manage vendor risks efficiently. It combines data-driven insights with predictive analytics to support smarter procurement and vendor selection decisions.

---

## 🎯 Problem Statement
Businesses often struggle to assess vendor reliability due to scattered data, lack of risk insights, and manual evaluation processes, leading to poor decision-making and potential financial loss.

---

## 💡 Solution
VendorSense provides a centralized platform that analyzes vendor data, predicts risks, and generates actionable insights using intelligent models and real-time analytics.

---

## 🧩 Key Features
- 🔍 Vendor Risk Prediction using ML models
- 📊 Data-driven insights and analytics dashboard
- 📄 Automated report generation (downloadable)
- 🤖 AI-powered assistant for decision support
- ⚡ Fast API backend with Flask
- 🎨 Modern frontend with dynamic UI

---

## 🏗️ Tech Stack

### Backend
- Python
- Flask API
- Machine Learning (Scikit-learn / Pandas)

### Frontend
- Next.js (App Router)
- TypeScript
