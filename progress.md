# 📈 VendorSense – Development Progress

**VendorSense** is a full-stack AI-driven vendor intelligence platform built to streamline and enhance supplier management. It leverages machine learning models to generate comprehensive vendor scores based on cost efficiency, delivery reliability, quality metrics, and overall performance. Beyond static evaluation, the system performs predictive risk analysis, estimating the likelihood of issues such as delays over a 6-month horizon using both historical data and external signals like market trends and sentiment.

The platform emphasizes explainable AI, breaking down every score and prediction into clear contributing factors so users can understand *why* decisions are made. It features an interactive dashboard with visual insights, vendor comparisons, and risk matrices, along with a conversational AI assistant that provides context-aware insights and actionable recommendations. Overall, VendorSense enables organizations to move from reactive vendor management to proactive, data-driven decision-making.
